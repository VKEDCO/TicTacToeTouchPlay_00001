1) Eclipse Project (ImageTouchDrive_00001.rar) with an Android 
application that shows how to use drawing touch driving to
the game of TicTacToe that the user can play with him/herself.
The application keeps track of the board state and informs
the user when X or O wins or there is a draw.

2) Targets: Android 2.3+, Eclipse Helios 3.6

3) To start: 
   a) unzip into a folder 
   b) open in Eclipse 
   c) Run as Android application
